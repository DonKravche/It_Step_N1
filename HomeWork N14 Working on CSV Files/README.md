# CSV File Processing Tasks

This project contains three tasks that involve reading, processing, and writing CSV files.

## Tasks

### Task 1: Titanic Survivors
- **Input File:** titanic.csv
- **Output File:** survived.csv
- **Description:** Reads the titanic.csv file and creates a new CSV file named "survived.csv" containing only the information of survivors.

### Task 2: Sort Organizations by Employee Count
- **Input File:** organizations-100.csv
- **Output File:** sorted_csv.csv
- **Description:** Reads the organizations-100.csv file, sorts the data based on the number of employees, and writes the sorted information to a new CSV file named "sorted_csv.csv".

### Task 3: SSL-Certified Companies
- **Input File:** organizations-100.csv
- **Output File:** ssl_companies.csv
- **Description:** Reads the organizations-100.csv file and creates a new CSV file named "ssl_companies.csv". It includes only the following information for companies with SSL-protected websites (HTTPS):
  - ID
  - Company Name
  - Website
  - Industry
  - Number of Employees
